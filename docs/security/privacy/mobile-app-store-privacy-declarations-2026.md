# Mobile App Store Privacy Declarations (2026)

Last updated: 2026-04-16

## Apple App Privacy

### Data Collection Summary

| Data Type | Collected | Linked to Identity | Used for Tracking |
| --- | --- | --- | --- |
| Contact Info (email) | Yes | Yes | No |
| Identifiers (user ID) | Yes | Yes | No |
| Usage Data (product interaction) | Yes | Yes | No |
| Diagnostics (crash data, performance) | Yes | No | No |

### Purpose of Data Collection

- **App Functionality**: User authentication, message delivery, agent interaction.
- **Analytics**: Aggregated usage patterns for product improvement (no third-party sharing).
- **Crash Reporting**: Diagnostic data for stability monitoring via Firebase Crashlytics.

### Data Retention

- Account data retained for the lifetime of the account plus 30 days after deletion request.
- Crash and diagnostic data retained for 90 days.
- Usage analytics aggregated and anonymised; raw events purged after 30 days.

### Data Sharing

No personal data is sold or shared with third parties for advertising or tracking purposes.

## Google Play Data Safety

### Data Collected

| Data Type | Collected | Shared | Purpose |
| --- | --- | --- | --- |
| Email address | Yes | No | Account management |
| User IDs | Yes | No | App functionality |
| App interactions | Yes | No | Analytics |
| Crash logs | Yes | No | App stability |
| Device or other IDs | Yes | No | Analytics |

### Security Practices

- Data encrypted in transit (TLS 1.2+).
- Data encrypted at rest (AES-256).
- Users can request data deletion via in-app settings or support contact.

### Data Deletion

Users may request deletion of their account and associated data through the app settings
or by contacting support@47network.com. Deletion is processed within 30 days.
