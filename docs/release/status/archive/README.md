# Sven Release Status Archive

This directory contains generated status artifacts that are no longer the canonical active view.

Examples:

- deprecated host-contract snapshots
- superseded `latest` outputs
- transitional environment captures

These files are kept for auditability only.

Use current active dashboards first:

- [../README.md](../README.md)

Host-specific archived `latest` snapshots live here:

- [host-specific-latest/README.md](host-specific-latest/README.md)

If an archived status file disagrees with a top-level `docs/release/status/*latest*` artifact or a canonical deploy doc, the archived file loses.
