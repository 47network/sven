# Agent Config Validator

Validates configuration files against schemas and detects configuration drift.

## Actions

- **create-schema**: Define a validation schema for a configuration type
- **validate-config**: Run a configuration file against its schema
- **detect-drift**: Compare current config against expected baseline
- **enforce-compliance**: Ensure configs meet organizational standards
- **audit-changes**: Track configuration changes over time
- **generate-template**: Create configuration templates from schemas
