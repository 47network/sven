# Data Catalog

Register, discover, and track data assets with automated lineage and profiling.

## Actions

### register-asset
Register a new data asset (table, view, stream, file, API, model, etc.)

### trace-lineage
Record data lineage between source and target assets

### profile-asset
Run automated data profiling (row counts, null rates, distributions)

### search-assets
Search the catalog by name, tags, type, or quality score

### score-quality
Calculate and assign a data quality score to an asset
