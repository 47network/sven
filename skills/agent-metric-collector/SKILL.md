# Agent Metric Collector

Collects metrics from Prometheus, StatsD, OpenTelemetry, and other sources with built-in alerting.

## Actions

- **add-source**: Register a new metric scrape target with interval config
- **query-metrics**: Query time-series data by name, labels, and time range
- **create-alert**: Define threshold-based alert rules with severity levels
- **analyze-trends**: Detect metric trends, anomalies, and forecast capacity
- **manage-dashboards**: Create and update metric visualization dashboards
- **export-metrics**: Export metric data to external monitoring systems
