# Agent Trace Analyzer

Configures distributed tracing, collects spans, and analyzes request flows for performance.

## Actions

- **configure-tracing**: Set up trace collection with sampling rate and propagation
- **query-traces**: Search traces by ID, operation, duration, or status
- **analyze-latency**: Identify latency bottlenecks across service boundaries
- **dependency-map**: Generate service dependency graphs from trace data
- **detect-anomalies**: Find unusual patterns in trace duration and error rates
- **export-traces**: Export trace data to Jaeger, Zipkin, or other backends
