# SIEM Connector

Security Information and Event Management integration with multi-source ingestion, event normalization, threat enrichment, and security dashboards.

## Actions

### ingest-events
Ingest security events from various sources with automatic normalization.

### enrich-event
Enrich a security event with threat intelligence and context data.

### create-dashboard
Create a security dashboard with customizable query definitions and visualizations.

### correlate-events
Run correlation queries across ingested events to identify threat patterns.

### export-report
Generate and export a security report for compliance or incident response.

### configure-source
Set up a new event source with parsing rules and normalization mapping.
