# Event Replayer

Replay event streams from any source with time-scaling, filtering, and checkpoint support.

## Actions

### replay-stream
Replay events from a source stream with configurable speed and filters.

### create-checkpoint
Save replay position for resumable sessions.

### filter-events
Apply include/exclude/regex filters to event streams.

### analyze-replay
Generate analytics from replay session results.

### compare-runs
Compare two replay sessions for divergence analysis.

### export-events
Export filtered events to external targets.
