## Actions
- **register-endpoint**: Register new service endpoint for discovery
- **resolve-endpoint**: Resolve service name to healthy endpoint URL
- **health-check**: Run health check against registered endpoints
- **configure-routing**: Set up routing rules (weighted, priority, canary)
- **deregister-endpoint**: Remove endpoint from service registry
- **list-services**: List all registered services and their endpoints
