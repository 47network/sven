# Certificate Authority

Manages the full certificate lifecycle including issuance, renewal, revocation, and compliance auditing for SSL/TLS and code signing certificates.

## Actions

### issue-certificate
Issue a new certificate for a domain or service.

### renew-certificate
Renew an expiring certificate with updated validity.

### revoke-certificate
Revoke a compromised or unused certificate.

### audit-certificates
Audit all certificates for expiry, compliance, and best practices.
