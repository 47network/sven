# Intrusion Guard

Real-time network intrusion detection and prevention system with signature-based and anomaly-based detection, auto-blocking, and event correlation.

## Actions

### create-rule
Define custom intrusion detection rules with pattern matching and protocol filters.

### monitor-traffic
Start real-time traffic monitoring with configured detection sensitivity.

### block-source
Manually block a source IP with configurable duration and reason.

### analyze-event
Deep analysis of a detected intrusion event with context and threat intelligence.

### whitelist-source
Add a trusted source to the whitelist to prevent false positives.

### export-events
Export intrusion events for SIEM integration or forensic analysis.
