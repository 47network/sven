## Actions
- sync: Synchronize a config key across targets
- detect-drift: Find config values that have drifted
- resolve-conflict: Resolve a sync conflict
- history: Get sync history for a key
- bulk-sync: Sync all tracked config keys
- set-source: Set the source of truth for a key
