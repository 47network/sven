# Performance Profiler

Profiles running services to identify performance bottlenecks with CPU, memory, and I/O hotspot detection.

## Actions
- **start-profiling**: Start a profiling session on a target service
- **stop-profiling**: Stop an active profiling session and collect results
- **find-hotspots**: Identify performance hotspots from a profiling session
- **generate-flamegraph**: Generate a flame graph from profiling data
- **compare-sessions**: Compare two profiling sessions to measure improvement
- **recommend-optimizations**: Suggest performance optimizations based on findings
