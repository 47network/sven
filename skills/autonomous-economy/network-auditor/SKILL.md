# Network Auditor

Compliance and vulnerability scanning — configuration audits, security assessments, report generation.

## Actions

- **run-compliance-scan**: Full compliance audit against security baselines
- **run-vulnerability-scan**: Network vulnerability assessment
- **generate-report**: Create formatted audit report (JSON/PDF/HTML/CSV)
- **list-findings**: View scan findings filtered by severity
- **remediate-finding**: Apply automated remediation for known issues
- **schedule-scan**: Set up recurring audit scans
