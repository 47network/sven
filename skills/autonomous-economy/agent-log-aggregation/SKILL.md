# Agent Log Aggregation & Search

Centralized log collection, indexing, searching and alerting across all agent activities.

## Actions

### create-stream
Create a new log stream for an agent or service.

### ingest-entry
Ingest a log entry into a stream with structured context.

### search-logs
Full-text and structured search across log entries with filters.

### build-dashboard
Create or update a log dashboard with customizable widgets.

### configure-alert
Set up alert rules based on log patterns or thresholds.

### export-logs
Export filtered log entries to external formats (CSV, JSON, NDJSON).

### generate-report
Generate a summary report of log activity, error rates and trends.
