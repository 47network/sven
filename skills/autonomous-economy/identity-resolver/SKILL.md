# Identity Resolver

Provides federated identity resolution across multiple identity providers. Supports user, service, and device identities with caching and audit logging.

## Actions
- **resolve-identity**: Resolve an identity from external provider
- **verify-identity**: Verify and validate an existing identity record
- **link-identity**: Link multiple identity records for cross-provider resolution
- **list-identities**: List all resolved identities with provider details
- **audit-access**: Generate identity access audit log
- **sync-provider**: Synchronize identity records from external provider
