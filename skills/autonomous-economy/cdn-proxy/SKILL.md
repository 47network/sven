# CDN Proxy

Manages content delivery with edge caching, compression, SSL termination, and cache invalidation for optimal content distribution.

## Actions

### configure-origin
Set up origin server connection with failover, timeouts, and retry policies.

### cache-warmup
Pre-populate edge caches with critical content for instant delivery.

### purge-cache
Purge cached content by path, prefix, tag, or complete cache flush.

### edge-analytics
Analyze cache hit rates, bandwidth savings, and per-location performance metrics.

### optimize-content
Configure content optimization including compression, image resizing, and minification.

### ssl-management
Manage SSL certificates, auto-renewal, and TLS configuration for edge endpoints.
