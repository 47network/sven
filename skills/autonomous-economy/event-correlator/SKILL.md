# Event Correlator

Intelligent event correlation engine that detects patterns across event streams, identifies root causes, and manages incidents with auto-learning capabilities.

## Actions

### create-pattern
Define a correlation pattern with event sequence and confidence threshold.

### detect-correlations
Run correlation detection across recent events within the configured time window.

### investigate-incident
Analyze a correlation incident to identify root cause and affected components.

### learn-patterns
Trigger auto-learning to discover new correlation patterns from historical data.

### resolve-incident
Resolve a correlation incident with resolution notes and learnings.

### export-patterns
Export discovered patterns for sharing or backup.
