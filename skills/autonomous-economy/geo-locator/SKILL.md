# Geo Locator

Performs IP geolocation lookups, enforces regional compliance zones, and manages data residency requirements.

## Actions

### lookup-ip
Resolve an IP address to geographic location data.

### check-compliance
Verify if a request meets regional compliance requirements.

### manage-restrictions
Create or update geo-restriction rules for a profile.

### generate-geo-report
Generate a geographic access pattern report.
