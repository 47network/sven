# IP Allocator

IPAM (IP Address Management) — pool creation, address allocation, conflict detection, audit logging.

## Actions

- **create-pool**: Define IP address pool from CIDR block
- **allocate-ip**: Assign IP address (static, dynamic, reserved, or floating)
- **release-ip**: Release allocated IP back to pool
- **detect-conflicts**: Scan for IP address conflicts
- **pool-utilization**: View pool usage and remaining capacity
- **audit-log**: View allocation history and changes
