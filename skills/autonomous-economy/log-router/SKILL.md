# Log Router

Routes, filters, and aggregates log streams across services with configurable pipelines.

## Actions

### create-pipeline
Create a new log processing pipeline with stages and routing rules.

### configure-filter
Set up log filtering rules based on level, source, and content patterns.

### route-logs
Route log entries to appropriate destinations based on configured rules.

### adjust-sampling
Modify sampling rates for high-volume log streams.

### analyze-throughput
Analyze pipeline throughput and identify bottlenecks.

### drain-pipeline
Gracefully drain a pipeline, processing remaining buffered entries.
