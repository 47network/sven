## Actions
- create-test: Define an A/B test with hypothesis and variants
- start: Begin running the test and collecting data
- record-event: Record an impression or conversion event
- get-results: Get current test results with p-value
- conclude: Conclude the test and declare a winner
- compare: Compare results across multiple tests
