# Metric Aggregator

Provides comprehensive metric collection, aggregation, and rollup capabilities for agent infrastructure monitoring.

## Actions
- **record-metric**: Record a single metric data point with dimensions
- **batch-record**: Record multiple metrics in a single operation
- **query-metrics**: Query metrics by name, time range, and dimensions
- **create-rollup**: Generate aggregated rollups for a time period
- **list-metrics**: List all tracked metric names with metadata
- **export-metrics**: Export metrics in Prometheus or OpenMetrics format
