# Secret Scanner

Continuous secret scanning — detect leaked credentials, API keys, and tokens across repositories, configs, and live environments.

## Actions

- **full-scan**: Scan all configured targets for exposed secrets
- **incremental-scan**: Scan only changes since last scan
- **verify-finding**: Confirm whether a detected secret is active/valid
- **remediate**: Auto-rotate or revoke detected exposed secrets
