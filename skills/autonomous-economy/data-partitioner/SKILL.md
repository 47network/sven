# Data Partitioner

Intelligent data partitioning — strategy planning, partition creation, rebalancing, and retention management.

## Actions

- **plan-partitions**: Analyze table and recommend optimal partitioning strategy
- **create-partitions**: Create partition scheme for a table
- **rebalance**: Rebalance data across partitions for even distribution
- **prune-old**: Remove expired partitions based on retention policy
