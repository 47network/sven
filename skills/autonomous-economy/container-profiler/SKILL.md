## Actions
- profile: Capture current resource usage snapshot
- analyze: Deep analysis of resource consumption patterns
- alert: Configure threshold-based alerting
- recommend: Generate right-sizing recommendations
## Inputs
- containerId: Target container to profile
- interval: Profiling interval in seconds
- thresholds: CPU and memory alert thresholds
- duration: Profiling duration
## Outputs
- cpuUsage: Current CPU utilization percentage
- memoryUsage: Current memory utilization percentage
- recommendations: Right-sizing suggestions
- alerts: Triggered threshold alerts
