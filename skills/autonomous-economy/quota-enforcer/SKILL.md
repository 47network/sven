# Quota Enforcer

Manages and enforces resource quotas with configurable policies, tracking, and violation handling.

## Actions

### set-quota
Define a resource quota with limits, period, and enforcement mode.

### check-usage
Check current usage against configured quota limits.

### enforce-policy
Apply enforcement actions when quotas are exceeded.

### reset-period
Reset usage counters for a new quota period.

### review-violations
Review and resolve quota violation records.

### adjust-limits
Dynamically adjust quota limits based on usage patterns.
