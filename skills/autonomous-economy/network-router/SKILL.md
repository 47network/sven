# Network Router

Manages network routing tables, traffic policies, and intelligent path selection for agent-to-agent and agent-to-service communication.

## Actions

### create-route
Create a new network route with destination, gateway, and metric configuration.

### update-policy
Create or update network policies for traffic filtering and forwarding rules.

### optimize-routing
Analyze traffic patterns and optimize routing tables for latency and throughput.

### health-check
Run health checks on all configured routes and update status accordingly.

### traffic-analysis
Generate traffic analysis reports with per-route statistics and anomaly detection.

### failover-config
Configure automatic failover routes for high-availability networking.
