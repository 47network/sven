# Metrics Hub

Centralized metrics collection and aggregation hub with Prometheus-compatible scraping, multi-format export, and alerting rule engine.

## Actions

### register-metric
Register a new metric series with type, unit, and label definitions.

### record-value
Record a metric value with optional labels and timestamp.

### create-alert-rule
Create threshold-based alerting rules with configurable duration and severity.

### query-metrics
Query metric series with label filtering and time range selection.

### export-metrics
Export metrics in Prometheus, OpenMetrics, JSON, or StatsD format.

### aggregate-window
Compute aggregated values over configurable time windows.
