# Security Scanner

Scans agent infrastructure, dependencies, configurations, and code for security vulnerabilities. Provides severity-rated findings with remediation guidance.

## Actions

- **create-profile**: Define a scan profile with targets and severity threshold
- **run-scan**: Execute a security scan against a profile
- **review-findings**: Review scan results with severity breakdown
- **apply-remediation**: Apply a remediation action for a finding
- **schedule-scan**: Set up recurring scans on a schedule
- **compare-scans**: Compare results between scan runs for trend analysis
