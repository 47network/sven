# Data Transformer

Transform data between formats with configurable mapping rules, field transformations, and parallel processing for high-throughput workloads.

## Actions

### transform_data
Transform records from source format to target format using configured rules.

### batch_transform
Process large datasets in parallel batches with progress tracking.

### validate_schema
Validate data against expected schema before transformation.

### create_rule
Create a new transformation rule with field mapping and expressions.

### preview_transform
Preview transformation output on a sample of records before full run.

### list_rules
List all configured transformation rules for a config.
