# Request Validator

API request validation — schema enforcement, input sanitization, rate limiting, and content-type policy enforcement.

## Actions

- **validate-schema**: Validate request payload against JSON Schema
- **sanitize-input**: Sanitize request inputs to prevent injection attacks
- **enforce-rate-limit**: Apply rate limiting rules per client/endpoint
- **audit-requests**: Audit request patterns and generate validation reports
