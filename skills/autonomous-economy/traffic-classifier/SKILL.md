# Traffic Classifier
Classify network traffic using DPI, heuristics, and ML-based methods.
## Actions
- classify-flow: Classify a specific network flow
- bulk-classify: Classify all flows on an interface
- update-signatures: Update classification signatures
- generate-report: Traffic composition report
- create-policy: Create classification-based policy
- export-results: Export classification results
