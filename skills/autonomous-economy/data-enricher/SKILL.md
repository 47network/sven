# Data Enricher

Enrich datasets by augmenting records with additional data from external APIs, databases, caches, and computed fields with batch processing and rate limiting.

## Actions

### enrich_records
Enrich a batch of records using configured sources.

### configure_source
Add or update an enrichment source with connection and mapping details.

### preview_enrichment
Preview enrichment results on a sample before full processing.

### run_enrichment_job
Start a full enrichment job with progress tracking.

### check_source_health
Verify connectivity and response from enrichment sources.

### list_sources
List all configured enrichment sources with status.
