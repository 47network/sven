# Agent Service Mesh & Discovery

Service registration, discovery, health monitoring and traffic management for the agent ecosystem.

## Actions

### register-service
Register a new service instance in the mesh with endpoints and health checks.

### discover-service
Find available service instances by name, protocol or tags.

### health-check
Run health checks against registered services and update status.

### configure-traffic
Set up load balancing, circuit breakers and retry policies.

### map-dependencies
Declare and visualize service dependency graphs.

### deregister-service
Gracefully remove a service from the mesh registry.

### mesh-report
Generate a service mesh topology and health summary report.
