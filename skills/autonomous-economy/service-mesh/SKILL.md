## Actions

### deploy-mesh
Deploy a new service mesh with mTLS and sidecar injection configuration

### register-service
Register a service with the mesh including health checks and traffic policies

### create-traffic-rule
Create traffic routing rules with weight-based splitting and retries

### configure-circuit-breaker
Set up circuit breaker policies for fault tolerance

### mesh-topology
View the full service mesh topology with connection status

### observability-dashboard
Generate observability metrics for all mesh services
