## Actions
- create-change: Create a change request with risk assessment
- approve: Approve or reject a change request
- implement: Begin implementing an approved change
- rollback: Execute rollback plan for a failed change
- audit: Get audit trail for a change
- schedule: Schedule a change for a future window
