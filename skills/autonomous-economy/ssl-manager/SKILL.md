## Actions
- **issue-certificate**: Request and issue SSL/TLS certificate for domain
- **renew-certificate**: Initiate certificate renewal process
- **audit-ssl**: Run SSL security audit and grade assessment
- **configure-protocols**: Configure supported TLS protocols and ciphers
- **monitor-expiry**: Check and alert on upcoming certificate expirations
- **revoke-certificate**: Revoke compromised or unused certificate
