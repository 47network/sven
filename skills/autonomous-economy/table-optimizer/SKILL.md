# Table Optimizer

Database table health management — bloat detection, vacuum scheduling, analyze runs, and index maintenance.

## Actions

- **health-check**: Assess table health including bloat, dead tuples, and index efficiency
- **schedule-vacuum**: Configure and run vacuum operations on bloated tables
- **recommend-indexes**: Analyze query patterns and recommend index improvements
- **defragment**: Rebuild fragmented tables and indexes for optimal performance
