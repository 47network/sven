# Dependency Scanner

Scans project dependencies for known vulnerabilities, outdated packages, and license compliance issues with automatic fix suggestions.

## Actions
- **full-scan**: Run a comprehensive dependency vulnerability scan
- **quick-scan**: Fast scan for critical and high severity issues only
- **check-updates**: Check for available dependency updates
- **auto-fix**: Automatically apply safe patches for known vulnerabilities
- **generate-sbom**: Generate Software Bill of Materials report
