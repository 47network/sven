# Format Converter

Convert files between data formats (JSON, CSV, XML, YAML, Parquet, etc.) with encoding handling, metadata preservation, and configurable field mappings.

## Actions

### convert_file
Convert a file from source format to target format.

### batch_convert
Convert multiple files in a batch operation.

### detect_format
Auto-detect the format of an input file.

### create_mapping
Create a reusable field mapping for format conversions.

### validate_output
Validate converted output against expected schema.

### list_mappings
List all configured format mappings.
