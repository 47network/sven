# Data Migrator

Plan and execute data migrations across database systems with validation and rollback.

## Actions

### create-plan
Create a migration plan with field mappings and transformations.

### validate-plan
Validate migration plan against source and target schemas.

### execute-migration
Execute a validated migration plan with progress tracking.

### rollback-migration
Roll back a failed or incorrect migration.

### compare-schemas
Compare source and target schemas for compatibility.

### export-report
Generate a migration execution report.
