# Data Archiver

Automated data archival — scheduled archiving, compression, cold storage management, and retention policy enforcement.

## Actions

- **archive-table**: Archive old records from a table to cold storage
- **verify-archive**: Verify archived data integrity and accessibility
- **restore-archive**: Restore archived data back to active database
- **enforce-retention**: Apply retention policies and purge expired archives
