# Token Rotator

Automated credential rotation — scheduled key rotation, grace periods, old-key revocation, and rotation health monitoring.

## Actions

- **rotate-now**: Immediately rotate a token with configurable grace period
- **schedule-rotation**: Set up automated rotation on a recurring schedule
- **check-health**: Assess token age and rotation readiness
- **revoke-expired**: Revoke tokens past their grace period
