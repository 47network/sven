# Session Recorder

Records agent interaction sessions with full input/output capture for debugging, training data extraction, and compliance auditing.

## Actions
- **record-session**: Start recording an agent session with configurable capture settings
- **replay-session**: Replay a recorded session for debugging or training
- **export-session**: Export session data in structured format for analysis
- **search-sessions**: Search recorded sessions by type, tags, or content
- **cleanup-expired**: Remove sessions past retention period
