# Latency Analyzer

Network latency measurement and analysis — target monitoring, baseline tracking, breakdown analysis.

## Actions

- **add-target**: Register host/port for continuous latency monitoring
- **measure-latency**: Run on-demand latency measurement with breakdown
- **compute-baseline**: Calculate P50/P95/P99 baseline from historical data
- **detect-degradation**: Compare current latency against baseline thresholds
- **traceroute**: Run hop-by-hop latency analysis
- **generate-report**: Latency trends and anomaly report
