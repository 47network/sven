## Actions

### create-vlan
Create a VLAN with ID, subnet, and gateway configuration

### assign-port
Assign a switch port to a VLAN as access or trunk

### create-acl
Create an access control list rule for inter-VLAN traffic

### vlan-topology
View VLAN topology with port assignments and ACL summary

### trunk-config
Configure trunk ports with allowed VLAN lists

### vlan-stats
View VLAN traffic statistics and port utilization
