# Connection Pooler

Database and service connection pool management — sizing, monitoring, drain/refill operations.

## Actions

- **create-pool**: Create connection pool for PostgreSQL, MySQL, Redis, HTTP, or gRPC backends
- **resize-pool**: Adjust min/max connections dynamically
- **drain-pool**: Gracefully drain connections for maintenance
- **pool-metrics**: View real-time pool utilization and hit rates
- **list-connections**: View active/idle connection details
- **health-check**: Verify backend connectivity and pool health
