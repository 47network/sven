## Actions

### add-target
Add a new endpoint to monitor with check type, interval, and expected response

### run-check
Immediately run a health check against a specific target

### list-incidents
View current and historical incidents for monitored targets

### configure-alerts
Set up notification rules for health check failures

### uptime-report
Generate uptime report with SLA compliance metrics

### pause-monitoring
Temporarily pause health checks for a target during maintenance
