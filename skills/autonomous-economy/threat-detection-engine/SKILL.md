# Threat Detection Engine
Monitors for security threats using configurable detection rules with automatic severity classification and response actions.
## Actions
### create-rule
Creates a threat detection rule with pattern matching, severity level, and response action.
### scan-now
Triggers an immediate threat scan across all monitored resources and surfaces.
### list-events
Lists detected threat events with filtering by severity, status, or time range.
### investigate-event
Retrieves detailed evidence and context for a specific threat event for analysis.
### update-sensitivity
Adjusts detection sensitivity level and auto-block configuration.
### resolve-event
Marks a threat event as resolved, mitigated, or false positive with resolution notes.
