# Config Auditor

Scans service configurations against compliance frameworks and best practices.
Detects misconfigurations, policy violations, and security gaps.

## Actions

### scan-configs
Run a full compliance scan across all managed services.

### create-baseline
Capture current configuration as the approved baseline.

### compare-baseline
Compare current config against baseline, report deviations.

### list-violations
List all active violations filtered by severity.

### remediate-violation
Apply recommended fix for a specific violation.

### generate-report
Produce a compliance report suitable for auditors.
