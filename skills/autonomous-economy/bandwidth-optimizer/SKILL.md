# Bandwidth Optimizer

Traffic shaping and QoS management — bandwidth allocation, burst control, utilization monitoring.

## Actions

- **create-profile**: Define bandwidth profile with limits and priorities
- **allocate-bandwidth**: Assign bandwidth to services, agents, or subnets
- **monitor-usage**: Real-time bandwidth utilization metrics
- **optimize-shaping**: Auto-tune shaping algorithm parameters
- **generate-report**: Bandwidth usage and efficiency report
- **set-burst-limit**: Configure burst allowances per profile
