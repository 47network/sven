# Index Optimizer

Analyzes query patterns and recommends optimal database indexes, identifies redundant indexes, and tracks performance impact of index changes.

## Actions
- **analyze-table**: Analyze a table's query patterns and recommend index improvements
- **apply-recommendations**: Apply recommended index changes with rollback support
- **find-redundant**: Identify redundant or unused indexes wasting storage
- **benchmark-impact**: Measure performance impact of index changes
- **generate-report**: Generate comprehensive index health report
