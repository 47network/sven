# Workflow Automator
Designs and executes complex multi-step workflows with conditional branching, error handling, and parallel execution capabilities.
## Actions
### create-workflow
Defines a new workflow with steps, triggers, and configuration.
### execute-workflow
Runs a workflow instance, tracking progress through each step.
### pause-workflow
Pauses a running workflow, preserving state for later resumption.
### resume-workflow
Resumes a paused workflow from its last completed step.
### get-workflow-status
Returns detailed status including current step, duration, and any errors.
### list-workflows
Lists all workflows with filtering by status, trigger type, and date range.
