# Feature Toggle

Manage feature flags with gradual rollout, targeting rules, and evaluation tracking.

## Actions

### create-flag
Create a new feature flag with default value and rollout rules.

### evaluate-flag
Evaluate a feature flag for a given context.

### update-rollout
Update rollout percentage or targeting rules.

### archive-flag
Archive an inactive feature flag.

### list-evaluations
List flag evaluation history.

### analyze-impact
Analyze feature flag impact on system behavior.
