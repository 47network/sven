# ACL Auditor
Audit ACL rules for shadowed entries, conflicts, and security gaps.
## Actions
- full-audit: Complete ACL audit
- shadow-check: Find shadowed rules
- conflict-detect: Detect rule conflicts
- optimize-rules: Suggest rule consolidation
- compliance-check: Verify against security policy
- export-findings: Export audit findings
