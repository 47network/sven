# Policy Engine

Manages security and governance policies that control agent operations. Evaluates actions against defined rules and enforces access, data, network, compliance, and operational policies.

## Actions

- **create-policy**: Define a new security policy with rules and enforcement mode
- **evaluate-action**: Evaluate an action against applicable policies
- **grant-exception**: Create a temporary policy exception with approval workflow
- **list-policies**: List all policies with filtering by type and status
- **audit-evaluations**: Review policy evaluation history and decisions
- **update-enforcement**: Change policy enforcement mode (enforce/audit/disabled)
