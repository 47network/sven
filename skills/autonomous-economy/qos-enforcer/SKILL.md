# QoS Enforcer
Enforce QoS policies with bandwidth allocation, DSCP marking, and scheduling.
## Actions
- configure-classes: Set up QoS traffic classes
- apply-policy: Apply QoS policy to interface
- monitor-queues: Monitor queue utilization
- detect-violations: Detect SLA violations
- adjust-bandwidth: Dynamic bandwidth adjustment
- generate-report: QoS compliance report
