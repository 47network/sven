# Firewall Policy Manager
Manage firewall rules with change tracking and approval workflows.
## Actions
- analyze-ruleset: Analyze current firewall rules
- propose-change: Propose rule change with justification
- apply-change: Apply approved rule change
- rollback-change: Rollback recent change
- audit-trail: View change history
- zone-review: Review zone-based security model
