# Packet Inspector

Deep packet inspection and analysis — capture management, anomaly detection, protocol breakdown.

## Actions

- **create-policy**: Define inspection policy with depth and protocol filters
- **start-capture**: Begin packet capture session
- **stop-capture**: End capture and generate summary
- **analyze-anomalies**: Review detected anomalies with severity classification
- **protocol-breakdown**: Get protocol distribution analysis
- **export-capture**: Export capture data for offline analysis
