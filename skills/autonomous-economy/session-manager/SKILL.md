## Actions
- **create-session**: Initialize new user or API session
- **refresh-session**: Extend session lifetime and update activity
- **terminate-session**: End active session and cleanup resources
- **apply-policy**: Configure session timeout and concurrency policies
- **audit-sessions**: Review active sessions for security compliance
- **bulk-expire**: Expire all sessions matching criteria
