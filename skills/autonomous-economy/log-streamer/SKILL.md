# Log Streamer

Provides centralized log streaming and aggregation for agent operations with real-time filtering, pattern-based alerting, and multi-destination routing.

## Actions

### create-stream
Create a new log stream with source type and format configuration.

### configure-alerts
Set up pattern-based log alerts with severity levels and cooldown periods.

### search-logs
Search across log streams with full-text and structured query support.

### export-logs
Export log data in various formats for analysis or archival.

### stream-stats
Get real-time statistics on log volume, error rates, and processing metrics.

### rotate-retention
Manage log retention policies and trigger manual rotation.
