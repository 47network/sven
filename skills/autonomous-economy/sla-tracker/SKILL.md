# SLA Tracker

Monitors SLA/SLO compliance with real-time error budget tracking, burn rate alerts, and violation history.

## Actions
- **create-objective**: Define a new SLO with target percentage and SLI type
- **record-measurement**: Record an SLI measurement against an objective
- **check-budget**: Check remaining error budget for an objective
- **detect-violations**: Detect and record SLA violations
- **calculate-burn-rate**: Calculate current error budget burn rate
- **generate-sla-report**: Generate a comprehensive SLA compliance report
