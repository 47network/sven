# Endpoint Monitor

Monitors endpoint availability with configurable health checks, alerts on downtime or degradation, and tracks uptime history.

## Actions

### add-endpoint
Register a new endpoint for monitoring with check interval and expectations.

### check-health
Run an immediate health check on a monitored endpoint.

### review-alerts
Review active and historical alerts for monitored endpoints.

### generate-uptime-report
Generate an uptime report with response time trends.
