# Log Analyzer

Performs intelligent log analysis including pattern detection, anomaly identification, and correlation across log sources.

## Actions
- **start-analysis**: Begin analyzing logs from specified source and time range
- **detect-patterns**: Identify recurring patterns and clusters in logs
- **find-anomalies**: Detect anomalous log entries using statistical analysis
- **correlate-logs**: Correlate log entries across multiple sources
- **search-logs**: Search logs using structured queries and filters
- **generate-report**: Generate a log analysis summary report
