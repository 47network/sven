# SSL Inspector

Comprehensive SSL/TLS inspection — certificate health, protocol analysis, vulnerability scanning, and compliance monitoring.

## Actions

- **scan-domain**: Full SSL scan of a domain including certificate chain, protocols, ciphers
- **check-expiry**: Monitor certificate expiry dates and alert on approaching deadlines
- **audit-compliance**: Verify SSL configuration against security standards (PCI-DSS, HIPAA)
- **vulnerability-scan**: Check for known SSL/TLS vulnerabilities (POODLE, Heartbleed, BEAST)
