# Compliance Reporter

Automates compliance reporting across multiple frameworks. Collects evidence, maps controls, scores compliance posture, and generates audit-ready reports.

## Actions
- **generate-report**: Generate a comprehensive compliance report for specified framework
- **collect-evidence**: Automatically collect evidence for control requirements
- **map-controls**: Map organizational controls to framework requirements
- **score-posture**: Calculate overall compliance score with breakdown
- **schedule-report**: Configure recurring compliance report generation
- **export-audit**: Export audit-ready documentation package
