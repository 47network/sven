# Incident Manager

Manages the full lifecycle of security incidents from detection through resolution and post-mortem. Coordinates response actions and tracks lessons learned.

## Actions

- **create-incident**: Open a new security incident with type and severity
- **respond**: Record a response action (investigate, contain, eradicate, recover)
- **escalate**: Escalate incident severity or assign to another responder
- **resolve**: Mark incident as resolved with resolution details
- **create-postmortem**: Generate post-mortem with root cause and action items
- **query-incidents**: Search incidents by type, severity, or status
