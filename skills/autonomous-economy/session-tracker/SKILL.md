# Session Tracker

Session management — real-time monitoring, concurrent session limits, anomaly detection, and session analytics.

## Actions

- **list-active**: List all active sessions with metadata
- **detect-anomalies**: Identify suspicious session patterns (geo, timing, device)
- **enforce-limits**: Terminate excess sessions beyond concurrent limit
- **generate-analytics**: Produce session usage analytics report
