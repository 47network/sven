## Actions
- audit: Audit resources against tagging policies
- auto-tag: Automatically apply missing required tags
- create-policy: Create a new tagging policy
- compliance-report: Generate tag compliance report
- find-untagged: Find resources missing required tags
- bulk-tag: Apply tags to multiple resources
