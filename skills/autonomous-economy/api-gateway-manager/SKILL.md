# API Gateway Manager

Manages API gateway routes, consumer API keys, rate limiting, and analytics for upstream service proxying.

## Actions

### create-route
Register a new API route with upstream, auth, and rate limiting config.

### manage-consumer
Create or update an API consumer with key and quota.

### analyze-traffic
Review API analytics — response times, error rates, traffic patterns.

### version-route
Create a new version of an existing API route.
