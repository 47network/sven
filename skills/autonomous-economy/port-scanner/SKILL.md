# Port Scanner

Network port scanning and service discovery — TCP/UDP scanning, service detection, OS fingerprinting.

## Actions

- **scan-target**: Run port scan against target host
- **detect-services**: Identify running services and versions
- **schedule-scan**: Set up recurring port scans
- **compare-results**: Diff scan results to detect changes
- **risk-assessment**: Evaluate port exposure risk levels
- **export-results**: Export scan results for reporting
