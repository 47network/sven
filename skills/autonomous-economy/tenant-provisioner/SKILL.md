# Tenant Provisioner

Provisions and manages isolated environments for multi-tenant agent workloads with configurable isolation levels and resource quotas.

## Actions
- **provision-tenant**: Create a new isolated tenant environment with resource allocation
- **deprovision-tenant**: Safely tear down a tenant and clean up resources
- **scale-tenant**: Adjust resource quotas for an existing tenant
- **migrate-tenant**: Move a tenant between isolation levels or infrastructure
- **audit-tenant**: Review tenant resource usage and compliance
