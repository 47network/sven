# Agent Cost Optimization

Budget management, spend tracking, cost forecasting and optimization recommendations for agents.

## Actions

### create-budget
Create a cost budget with period, amount and alert thresholds.

### record-spend
Record a cost entry against a budget for compute, LLM tokens or other resources.

### forecast-spend
Generate a cost forecast for a budget period using historical data.

### recommend-savings
Analyze spending patterns and recommend cost reduction strategies.

### check-alerts
Review triggered cost alerts and acknowledge or escalate them.

### budget-report
Generate a comprehensive cost and utilization report.

### optimize-resources
Apply accepted recommendations to optimize resource allocation.
