# Dashboard Builder

Create and manage real-time monitoring dashboards with customizable panels, multiple data sources, and shareable snapshots.

## Actions

### create-panel
Add a new panel to a dashboard with chart type, query, and layout configuration.

### configure-layout
Arrange dashboard panels using grid, freeform, or row/column layouts.

### create-snapshot
Capture a point-in-time snapshot of all dashboard panels for sharing.

### set-refresh
Configure auto-refresh intervals and default time ranges.

### share-dashboard
Generate shareable URLs for dashboard snapshots with expiration.

### import-template
Import pre-built dashboard templates for common monitoring scenarios.
