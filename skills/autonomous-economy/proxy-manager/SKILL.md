# Proxy Manager

Reverse and forward proxy orchestration — route management, caching, SSL termination, load distribution.

## Actions

- **create-proxy**: Configure reverse/forward proxy with upstream targets
- **manage-rules**: Add rewrite, redirect, block, and rate-limit rules
- **view-logs**: Access proxy access logs with filtering and aggregation
- **health-check**: Run upstream health verification
- **toggle-cache**: Enable/disable response caching per route
- **update-ssl**: Configure SSL certificates for proxy endpoints
