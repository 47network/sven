# Edge Router

Edge routing and traffic management — TLS termination, path-based routing, caching, rate limiting.

## Actions

- **create-config**: Set up edge router configuration with upstream targets
- **add-route**: Define path-based routing rules with priorities
- **configure-tls**: Set TLS mode (terminate, passthrough, mutual)
- **enable-cache**: Configure response caching per route
- **set-rate-limit**: Apply rate limiting to routes
- **view-access-logs**: Query edge access logs with filtering
