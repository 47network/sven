# VPN Provisioner

WireGuard and OpenVPN tunnel provisioning — peer management, key rotation, connection monitoring.

## Actions

- **provision-tunnel**: Create WireGuard/OpenVPN server configuration
- **add-peer**: Register new VPN peer with key generation
- **rotate-keys**: Rotate private/preshared keys for security
- **monitor-connections**: View active connections and transfer stats
- **revoke-peer**: Disable and remove a VPN peer
- **export-config**: Generate downloadable client configuration files
