# Proxy Configurator

Automated reverse proxy setup — upstream configuration, routing rules, caching policies, and health monitoring.

## Actions

- **configure-upstream**: Set up upstream targets with health checks and load balancing
- **create-route**: Define routing rules with path matching, header rewriting, and timeouts
- **setup-caching**: Configure response caching policies per route
- **health-check**: Monitor upstream health and trigger failover
