# Cache Warmer

Proactively populate and refresh caches using predictive and scheduled strategies.

## Actions

### warm-cache
Populate cache entries based on configured strategy.

### analyze-hit-rate
Analyze cache hit/miss rates and suggest optimizations.

### configure-ttl
Set and optimize TTL values per cache key pattern.

### evict-stale
Remove stale or low-priority cache entries.

### predict-access
Use access patterns to predict and pre-warm likely needed entries.

### export-stats
Export cache performance statistics.
