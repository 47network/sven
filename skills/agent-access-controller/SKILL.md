# Agent Access Controller

Manages access policies, role definitions, permission grants, and scope enforcement across the platform.

## Actions
- **create-policy**: Define access policy (RBAC, ABAC, ACL, rule-based)
- **define-role**: Create roles with permissions and scope
- **grant-access**: Assign roles to subjects (agents, users, services)
- **revoke-access**: Revoke access grants with reason tracking
- **evaluate-access**: Check if subject has permission for resource
- **audit-grants**: Generate access audit reports with compliance checks
